# 工作流（Workflow）



## 概念

把业务之间的各个步骤以及规则进行抽象和概括性的描述。

使用特定的语言为业务流程建模，让其运行在计算机上，并让计算机进行计算和推动。



## BPMN

业务流程模型和标记法。

工作流是对各个行业的流程进行抽象，例如电信、软件、制造、金融和办公自动化领域。为了给全部业务的参与者提供易于理解的标准标记法，由业务流程管理倡议组织 BPMI 开发出了**业务流程建模标记法** （Business Process Modeling Notation, BPMN）。
